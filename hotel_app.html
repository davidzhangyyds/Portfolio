Project Presentation

Introduction

The aim of the project is to develop a simplified management application for a hotel called R&T. The application will allow customers to make reservations by providing their information and preferences, while the hotel will be able to manage these reservations and calculate the associated costs. The project uses the Python language, the Kivy (or Kivymd) graphical interface, CGI-Python for server-side processing, and a MySQL or MariaDB DBMS for database management.
