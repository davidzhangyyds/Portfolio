Introduction

The company has several networks where machines receive IP addresses via DHCP. However, in order to restrict access to the network, only machines whose MAC address is registered by the network and system department can obtain an address via DHCP, and this address is fixed. In order to centralise the management of the DHCP servers on each network, the company wants to install a central server that can access these different DHCP servers. From services hosted on the network machines also needs to be managed centrally. centralised. The general objective is therefore to have a central machine that supervises the DHCP servers on each network and administers the services running on the various machines on the local networks. It is therefore a DHCP supervision and service supervision tool. services.
